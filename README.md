# 北京航空航天大学 Beamer 模板 (BUAA Beamer Theme)

这是一个适用于北京航空航天大学（BUAA）学术报告和课堂演示的非官方 Beamer 模板。

## ✨ 特点

* [简洁]
* [现代]
* [符合北航 VI 规范]

## 🔧 如何使用

1.  **主文件:** `slide.tex`
2.  **编译方式:** 推荐使用 **[在这里填入你推荐的编译器，例如 XeLaTeX]** 进行编译，以确保中文字体和 Logo 正常显示。
3.  **参考文献:** 参考文献格式在 `ref.bib` 中管理。
4.  **图片:** 报告中使用的图片请放入 `pic` 文件夹。

## ⚙️ 自定义

* 修改 `slide.tex` 文件开头的 `\title`, `\author`, `\institute`, `\date` 来更新你的报告信息。
* Logo 文件位于 `BUAA-logo.png`，可替换为你需要的 Logo。
